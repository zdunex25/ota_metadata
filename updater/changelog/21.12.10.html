CHANGELOG 21.12.10


REUPLOAD 18th December
Fix - Some apps lagging on Android 12
Fix - Silent mode toggle doesn't work


Tip for Android 12 devices: Because TWRP doesn't work with Android 12, the update needs to be flashed in Fastboot mode. Download the ZIP package, unpack it and run update script (without formatting data if you're moving from Android 11 xiaomi.eu ROM or with format data if you're flashing it first time).

This is the last MIUI 12.5 Beta ROM. Starting from December 13th, the weekly updates will be temporarily suspended. Next updates will be based on MIUI 13 once it's ready.


This changelog doesn't mention all changes made by Xiaomi.eu or Xiaomi.
