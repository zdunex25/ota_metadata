CHANGELOG 22.2.23


System
New - Xiaomi.eu weekly ROM for Mi 11 LE/Xiaomi 11 Lite 5G NE (lisa)
Optimization - Updated Android Security Patch to February 2022
Optimization - You can press and hold the sidebar indicator to move it while holding device vertically
Fix - Certain third-party apps were not working correctly
Fix - Google app wasn't being set as default app for "Assist & voice input" automatically

Settings
Fix - Custom variable fonts were not listed in Font settings
Fix - Screen usage stats were missing in Battery settings [Android 12 devices]

Warning: Camera doesn't work properly on Mi 10T Pro. We don't recommend upgrading to this version if you own Mi 10T Pro. Redmi K30S Ultra/Mi 10T camera works fine.

Tip for Android 12 devices: Because TWRP doesn't work with Android 12, the update needs to be flashed in Fastboot mode. Download the ZIP package, unpack it on your PC and run update script (without formatting data if you're moving from Android 11 xiaomi.eu ROM or with format data if you're flashing it first time).

This changelog doesn't mention all changes made by Xiaomi.eu or Xiaomi.
