CHANGELOG 22.1.5


System
Optimization - Moved "System-wide variable font" to Developer options
Fix - Holding the power button to open the power menu takes 3 seconds [Android 12 only]

Notes
Fix - Mind maps were lost after disconnecting from the Xiaomi Cloud sync

Warning: Camera doesn't work properly on Mi 10T Pro. We don't recommend upgrading to this version if you own Mi 10T Pro. Redmi K30S Ultra/Mi 10T camera works fine.

Tip for Android 12 devices: Because TWRP doesn't work with Android 12, the update needs to be flashed in Fastboot mode. Download the ZIP package, unpack it and run update script (without formatting data if you're moving from Android 11 xiaomi.eu ROM or with format data if you're flashing it first time).

This changelog doesn't mention all changes made by Xiaomi.eu or Xiaomi.
