CHANGELOG 21.5.20


System
New - Xiaomi.eu TWRP ROM for Mi 11 (venus) and Mi 11 Pro/Ultra (mars/star). TWRP will persist after ROM install. To install this version, download specific TWRP image for your device, boot it using "fastboot boot twrp.img" and install ROM (do format data first if you didn't install xiaomi.eu before)

Lock screen
New - You can choose whether your fingerprint will be used on the Lock screen [Mi 9 Pro 5G, Redmi K30, Mi 10 Pro, Mi 10S, Mi 11, Mi 10T/Pro, Redmi K40/POCO F3, Mi 11 Pro/Ultra]
Fix - Couldn't open Face unlock settings occasionally [Mi 11 Pro/Ultra, Mi 11, Mi 10S, Mi 9]

Notification shade
Fix - Couldn't dismiss media playback notification for some apps [e.g. Spotify]

System launcher
Fix - Couldn't open the Calendar app from a clock widget by tapping a date

Calendar
Fix - Some settings, like "Start of the week", had no effect

Mail
New - You can configure notifications for selected folders now
Optimization - Office 365 login optimizations
Optimization - App settings changes

Camera
Fix - Quick snap in Video mode didn't work properly

This changelog doesn't mention all changes made by Xiaomi.eu or Xiaomi.
